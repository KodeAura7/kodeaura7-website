02

Typography

Display

Syne

Headlines & numbers. Weights 500–600, set tight (tracking-tighter). Also used light/extralight on card titles.

Body

Inter

Paragraphs, labels, UI. Weights 400–500. The workhorse — neutral so Syne can carry personality.

Utility

JetBrains

Eyebrows, tags, data. Set uppercase with tracking-widest for technical signposting.

Type scale

Hero headline text-5xl→7xl · Syne 600
Section heading text-3xl→5xl · Syne 600
Card title text-2xl · Syne 300
Body copy — the default reading size. text-base · Inter 400
Secondary / supporting text. text-sm · Inter 500
Eyebrow label text-xs · JetBrains Mono
